# Python Code Snippets Repository

A collection of standalone Python files, each solving a specific algorithm or data structure problem (e.g., `3Sum.py`, `Add_Digits.py`).

## Contents
- **Python Code Snippets/**: All problem-specific Python files.
- **Ground Truth of Python Code Snippets with (PEP 8 and Coding Best Practice Violations).docx**: Reference document outlining style and best-practice issues in each snippet.

## Purpose
The repository is intended for:
- Study Python code quality and PEP 8 compliance.
- Support research in static analysis and automated code review.
- Provide examples of common coding interview problems.

## Access
All snippets are (available upon request) and can run independently.
